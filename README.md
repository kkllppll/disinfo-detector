# DisInfo Detector

Програмне забезпечення автоматичного виявлення джерел дезінформації та не автентичної поведінки членами скоординованих груп.

---

## Опис проєкту

Система виконує збір новин з RSS-джерел, аналізує текстовий контент, обчислює семантичну схожість між статтями, формує події та виявляє групи джерел із ознаками скоординованої інформаційної активності.

Проєкт розроблений як дослідницький прототип у межах магістерської роботи.

---

## Основні можливості

- автоматичний збір новин через RSS
- NLP-векторизація тексту
- аналіз семантичної схожості
- використання кількох метрик:
  - cosine similarity
  - euclidean distance
  - manhattan distance
- кластеризація подій
- виявлення скоординованих груп
- графовий аналіз зв’язків між джерелами
- інтерактивна D3.js візуалізація
- аналітична веб-панель

---

---

## Структура проєкту

```text
collector/   - збір та парсинг RSS-джерел
processor/   - NLP обробка та векторизація
analytics/   - кластеризація та графовий аналіз
dashboard/   - веб-інтерфейс та візуалізація
```



## Pipeline аналізу

### Збір статей з RSS-джерел

```bash
python manage.py collect_news
```

### NLP-обробка та векторизація

```bash
python manage.py process_articles
```

### Кластеризація та виявлення груп

```bash
python manage.py cluster_articles
```

### Побудова графового аналізу

```bash
python manage.py build_graph
```

---

## Візуалізація

Система містить інтерактивну графову візуалізацію на базі D3.js для відображення семантичних зв’язків між джерелами та виявлення скоординованої активності.

---

## Призначення

Прототип демонструє архітектуру та аналітичні методи програмного забезпечення для виявлення дезінформації, координації інформаційної активності та аналізу мереж поширення контенту.
